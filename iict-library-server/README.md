# IICT Library Management System - Server

This folder contains the backend application built with Express.js, TypeScript, and Prisma.

## Getting Started

1.  Navigate to this directory: `cd iict-library-server`
2.  Install dependencies: `npm install`
3.  Set up your `.env` file by copying `.env.example`.
4.  Initialize the database: `npx prisma migrate dev --name init`
5.  Run the development server: `npm run dev`
